## UI / React Terms
| Term | Definition |
|------|-----------|
| Page component | Top-level component rendered for a route |
| Feature component | Component implementing feature-specific business logic |
| Common component | Reusable, presentational component (Button, Input, Modal) |
| Layout component | Structural component (Header, Sidebar, Footer) |
| Custom hook | Reusable stateful logic extracted into a `use*` function |
| Error boundary | Class component that catches render errors in children |
| Server state | Data fetched from an API, managed by a data-fetching library |
